Наш чек-лист по настройке окружения для работы с TypeScript выглядит так:

- Командой npm i -D typescript установить компилятор TypeScript.
- Проверить версию компилятора командой npx tsc -v.
- Создать конфигурационный файл tsconfig.json, выполнив команду npx tsc --init.
- В параметрах include и compilerOptions.rootDir указать, где будут расположены исходные файлы с кодом на TypeScript.
- В параметре compilerOptions.outDir указать, куда компилятору складывать итоговые файлы с кодом на JavaScript.
- При необходимости указать другие параметры компиляции.
- Выполнить компиляцию командой npx tsc.
